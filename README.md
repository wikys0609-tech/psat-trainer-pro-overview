# 📁 [프로젝트 이름] (예: PSAT Trainer Pro)

> **한 줄 소개:** 10년 차 출제 위원의 노하우를 담은 PSAT 실전 트레이닝 웹 앱
>
> **Live Site:** [https://psat-trainer-pro.vercel.app/]

## 💡 프로젝트 개요 (Overview)
* **개발 기간:** 2025.04 ~ 2025.05 (약 3주)
* **개발 인원:** 1인 개인 프로젝트
* **프로젝트 목적:** 기존 NCS/PSAT 수리능력 학습 방식의 비효율성을 개선하고, 약점 분석 기능 제공

## 🛠️ 기술 스택 (Tech Stack)
* **Frontend:** Next.js, React, Tailwind CSS
* **Backend:** Node.js, PostgreSQL
* **Deployment:** Vercel

## 📌 주요 기능 (Key Features)
1.  **훈련 모드:** 사칙 연산, 응용 연산, PSAT 쪼개 풀기 연습
2.  **실전 모드:** 상기 연습의 타임 어택 모드
3.  **성적 대시보드:** 리더보드
